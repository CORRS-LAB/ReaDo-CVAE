Only `demo_results.csv` is available on Github. Other two are larger than 80MB, available on [Hugginface-reado-cvae](https://huggingface.co/datasets/martingalefan/reado-cvae).

```
data/
│   ├── [ ] demo_data.h5ad                # small example dataset (~1000 cells, 3000 HVGs)

│   └── [x] demo_results.csv              # pre‑computed algorithm predictions (for CCS/ACP demo)
```
